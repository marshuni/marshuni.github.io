莫里斯 / Maurice
2023-01-29 / / ★★★★★ 力荐
干净的色调、康桥河上悠悠的小船、在阳光下晶莹透亮的短发、青草汁液的淡淡香气……两人探索着对彼此的爱情。电影前段如同一篇清新隽永的诗歌，缓缓揭开一段朦胧的情愫，却又在呼之欲出之时紧紧克制住。 两人最终分道扬镳。克莱夫抑制了内心的情感，在名利场打拼出一番事业，有了妻子，大抵是快意人生，却在关窗时望向窗外，恍惚间有些“夜深忽梦少年事”的怅然。 而莫里斯在迷惘和痛苦中挣扎许久，最终决定直面自我，心甘情愿放弃一切，跨越阶级走向一场精神与肉体合一的、纯粹的爱情。故事在这里聪明地画上了句号，把矛盾都推到未来，只给观众留下美好的遐想。 大概是受限于对当时的文化不够了解，后面亚力克的剧情没有太看懂，，因此不过多评价了。但含蓄优雅的绅士风度，和危险炽烈的感情交织在一起，就已足够让人沉醉其中。
满江红
2023-01-23 / / ★★★★☆ 推荐
前段充斥着冰冷阴沉的色调，着重营造“朔气传金柝，寒光照铁衣”的军营氛围和紧张压抑之感，照应着奸佞当道、人人自危的背景；而高墙之间狭窄的通道以及迷宫般的院落，仿佛也折射着人物们的城府之深。一出威逼奸臣秦桧的戏码，几个人层层叠叠的计谋，加起来都不虽然悬疑情节着实扣人心弦，但两个半小时的工于心计，各种略离谱的身份嵌套与反转，还是让人感觉有些疲惫。 纵使明枪暗箭皆不敌秦桧，在片尾桃丫头浅浅吟出“怒发冲冠”、全军慷慨应和的那一刻，“精忠报国”四字便已镌刻所有人的心中，秦桧也注定背负万世的骂名——这就是电影的主旨所在，铭记历史。影片巧妙地把主旋律融进对历史故事的新编中，既立足浓墨重彩的历史，也弘扬主旋律，同时完全脱离了单一的说教和歌颂，不失为一部以文化动人的春节佳作。
布达佩斯大饭店 / The Grand Budapest Hotel
2023-01-15 / / ★★★★★ 力荐
朦朦胧胧的胶片质感、真假参半的布景和分章节叙事的结构，似乎让我忘记了这部影片的拍摄时间，真正跌入那个蒙尘的曾经辉煌的时代。浓郁活泼的色调、儒雅随和的绅士风度、在监狱门口的吟诗和香水……以及那些闪耀着的浪漫的人性光芒，它们或许从未面对贪婪丑恶者低头，但最终在嘈杂纷争的时代中被吞噬殆尽。仅存“破败的钢琴在荒郊野岭的废墟中，勉强奏出的悲怆终曲”，黑色而幽默。
情书 / Love Letter
2023-01-06 / / ★★★★★ 力荐
你好吗？我很好。——博子在对恋人往昔的探索中渐渐接受了没有他的现实。 而女孩藤井树蓦然回首，白色窗帘被风吹起，藏着的竟是青涩的柔情。 怅然，但也沉醉其中。
